Picodroid is a stripped-down Android-shaped runtime that fits on a Raspberry Pi Pico. App code is written in Java, compiled to bytecode, and interpreted by a lightweight JVM built in Rust running on bare-metal embedded hardware.

If you're new, follow this path: [Build & flash](/get-started/build/) → [Your first app](/get-started/first-app/) → [Multi-screen tutorial](/tutorials/multi-screen-app/) → [Embedded gotchas](/guides/embedded-gotchas/) → [Limits & memory budgets](/reference/limits/). Then copy whichever [Example](/examples/) is closest to what you want to build.
